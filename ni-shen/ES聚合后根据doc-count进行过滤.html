ES聚合后根据 doc_count 进行过滤

2022-08-22
Word count: 218   |   Reading time≈ 1 min

语句解读：

搜索从当前开始100分钟内，匹配当前uri的数据，针对用户名做聚合，将聚合条数大于8的用户展示出来。

GET /nginx-*/_search
{
  "size": 0, 
  "query": {
    "bool": {
      
      "filter": [
        {
          "range": {
            "@timestamp": {
              "gte": "now-100m"
            }
          }
        }
      ], 
      
      "should": [
        {
            "match_phrase": 
        {
            "match_phrase": {
              "request_uri.keyword": "/zh-CN/account/login"
            }
        }        
      ],
      "minimum_should_match": 1
    }
  },
    "aggs": {
    "NAME": {
      "terms": {
        "field": "Eaa-User.keyword",
        "size": 1000
      },
            "aggs":{
        "having": {
          "bucket_selector": {
            "buckets_path": { 
              "view_count": "_count"
            },
            "script": "params.view_count > 8"
          }
        }
      }
    }
  }
}
